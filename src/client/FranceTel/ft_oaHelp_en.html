Business Livebox activation

The Business Livebox activation page is used to configure the username and password for the PPP connection established on the ADSL link.

Network access parameters

  • The Username field comprises 1 to 64 characters.
  • The Password field comprises 1 to 60 characters.
  • The PPP connection status parameter indicates whether the connection is UP or DOWN. To update this status the user must use the "Refresh" button in their browser. When the username or password are updated the status will probably remain DOWN after the page is refreshed and will take several seconds to change to UP.
  • The apply button establishes the PPP connection with the defined parameters.
  • The cancel button resets to default the values entered in the Username and Password fields.
  • When the username or password are updated, the message "Operation successful" is displayed in the right side of the window.
